# 🚀 Portfólio de Bruno Fernandes - Edição Premium

Este portfólio é um ecossistema digital de alta performance, projetado para demonstrar **autoridade técnica em Gestão de Projetos de TI e IA Generativa**. O projeto foca na aplicação estratégica de tecnologias modernas para criar experiências imersivas e funcionais.

![Preview do Portfolio](Imags/Bruno_Fernandes.jpeg)

## ✨ Destaques de Tecnologia & Inovação

### 🎙️ Modo Jarvis (Assistente de Voz Global)
Uma interface conversacional futurista que permite controlar o site por voz:
- **Comandos Globais**: "Abrir currículo", "Ir para projetos", "Contatar Bruno" ou "Alternar tema".
- **Feedback Vocal (TTS)**: O site responde vocalmente confirmando comandos, elevando a acessibilidade e o fator "WOW".
- **Interface Hands-Free**: Navegação fluida sem necessidade de cliques.

### 🧠 Bruno-bot 2.0 (Agente de IA)
Um assistente virtual inteligente integrado que guia a experiência do usuário:
- **Navegação Inteligente**: Responde perguntas sobre o currículo e navega automaticamente para as seções relevantes.
- **Neural Chips**: Sugestões de perguntas dinâmicas para engajamento imediato.
- **Ações Contextuais**: Abre certificados PDF e documentos diretamente via chat.

### 🌌 Neural Flow Field (Vortex 3D)
Uma experiência visual imersiva no Hero utilizando **Three.js**:
- **Campo de Fluxo Orgânico**: Nuvem de partículas com conexões dinâmicas (Plexus).
- **Adaptive Theme**: O cenário 3D detecta e altera cores automaticamente entre os modos Claro e Escuro.

### 📱 PWA (Progressive Web App)
O portfólio agora é um aplicativo instalável:
- **Offline Ready**: Funciona mesmo sem conexão estável através de Service Workers.
- **Instalável**: Pode ser adicionado à tela inicial do celular ou desktop como um app nativo.

### 📊 GitHub Real-time Insights
Validação de autoridade técnica através de dados reais de contribuição e repositórios.

## 🛠️ Stack Tecnológica

- **Core**: HTML5, CSS3 (Vanilla), JavaScript (ES6+).
- **Animações**: [GSAP](https://greensock.com/gsap/) (ScrollTrigger, Spring physics).
- **3D Engine**: [Three.js](https://threejs.org/).
- **PWA**: Service Workers & Manifest.json.
- **Voice**: Web Speech API (STT/TTS).
- **Scroll**: [Lenis](https://github.com/darkroomengineering/lenis) (Smooth scroll).
