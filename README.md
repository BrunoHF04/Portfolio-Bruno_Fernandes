# 🚀 Portfólio de Bruno Fernandes - Edição Premium

Este portfólio é um ecossistema digital de alta performance, projetado para demonstrar **autoridade técnica em Gestão de Projetos de TI e IA Generativa**. O projeto foca na aplicação estratégica de tecnologias modernas para criar experiências imersivas, estáveis e globais.

![Preview do Portfolio](Imags/Bruno_Fernandes.jpeg)

## ✨ Experiências & Tecnologias Premium

### 🌍 Global Localization (Suporte Trilíngue)
Sistema dinâmico de tradução que posiciona o portfólio para o mercado internacional:
- **Idiomas**: Português, Inglês e Espanhol.
- **Transições Fluidas**: Troca de idioma em tempo real utilizando animações GSAP para uma experiência de usuário sem interrupções.

### 🎵 Gold Ghibli Player (Trilha Sonora Personalizada)
Uma experiência de áudio imersiva que substitui integrações genéricas por uma solução artesanal:
- **YouTube Iframe API**: Integração estável com bypass de restrições de produção através de overlays interativos.
- **Aesthetic Ghibli**: Animações de vinil, barras de áudio em tempo real e efeito "Golden Glow" ativado pelo som.
- **Bypass Técnico**: Solução customizada para contornar políticas de Autoplay e CORS de navegadores modernos.

### 🎙️ Modo Jarvis (Assistente de Voz Global)
Uma interface conversacional futurista que permite controlar o site por voz:
- **Comandos Globais**: "Abrir currículo", "Ir para projetos", "Contatar Bruno" ou "Alternar tema".
- **Feedback Vocal (TTS)**: O site responde vocalmente confirmando comandos, elevando a acessibilidade e o fator "WOW".

### 🧠 Bruno-bot 2.0 (Agente de IA)
Assistente virtual inteligente integrado que guia a experiência:
- **Navegação Inteligente**: Responde perguntas sobre o currículo e navega automaticamente para as seções relevantes.
- **Neural Chips**: Sugestões de perguntas dinâmicas e acesso direto a certificados PDF via chat.

### 🌌 Neural Flow Field (Vortex 3D)
Experiência visual no Hero utilizando **Three.js**:
- **Campo de Fluxo Orgânico**: Nuvem de partículas distribuída de forma matemática com conexões Plexus.
- **Adaptive Theme**: Cenário 3D com troca automática de paleta entre os modos Claro e Escuro.

### 📊 GitHub Real-time Insights
Validação de autoridade técnica através da integração direta com a API do GitHub para métricas de contribuição, commit e linguagens em tempo real.

## 🛠️ Stack Tecnológica

- **Core**: HTML5, CSS3 (Vanilla Premium), JavaScript (ES6+).
- **Animações**: [GSAP](https://greensock.com/gsap/) (ScrollTrigger, Spring physics, Text scramble).
- **Audio Engine**: [YouTube Iframe API](https://developers.google.com/youtube/iframe_api_reference) com ativação por overlay.
- **3D Engine**: [Three.js](https://threejs.org/).
- **PWA**: Service Workers & Manifest.json para funcionamento Offline.
- **Voice**: Web Speech API (STT/TTS).
- **Scroll**: [Lenis](https://github.com/darkroomengineering/lenis) (Smooth scroll).
