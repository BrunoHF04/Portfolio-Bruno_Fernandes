# 🚀 Portfólio de Bruno Fernandes - Edição Premium

Este portfólio é um ecossistema digital de alta performance, projetado para demonstrar **autoridade técnica em Gestão de Projetos de TI e IA Generativa**. O projeto vai além do visual, integrando agentes inteligentes, simulações 3D e prova social baseada em dados reais.

![Preview do Portfolio](Imags/Bruno_Fernandes.jpeg)

## ✨ Destaques de Tecnologia & Inovação

### 🧠 Bruno-bot 2.0 (Agente de IA)
Um assistente virtual inteligente integrado que guia a experiência do usuário:
- **Navegação Inteligente**: Responde perguntas sobre o currículo e navega automaticamente para as seções relevantes.
- **Neural Chips**: Sugestões de perguntas dinâmicas para engajamento imediato.
- **Ações Contextuais**: Abre certificados PDF e documentos diretamente via chat.
- **Google Search Fallback**: Integração inteligente para buscas externas quando o robô não possui a resposta técnica local.

### 🌌 Neural Flow Field (Vortex 3D)
Uma experiência visual imersiva no Hero utilizando **Three.js**:
- **Campo de Fluxo Orgânico**: Partículas que se movem de forma fluida simulando uma rede neural.
- **Conexões Dinâmicas (Plexus)**: Links que se formam e rompem em tempo real entre os dados.
- **Adaptive Theme**: O cenário 3D detecta e altera cores e comportamentos automaticamente entre os modos Claro e Escuro.
- **Paralaxe Interativo**: Profundidade extrema que reage ao movimento do mouse do usuário.

### 📊 GitHub Real-time Insights
Validação de autoridade técnica através de dados:
- **Real-time Heatmap**: Representação visual do gráfico de contribuições do GitHub.
- **Dynamic Stats**: Contadores automáticos de repositórios e commits via API REST.

### 🎨 Design System: Premium Glassmorphism
- **Estética Ultra-Clean**: Refinamento de transparências, desfoques (blur) e ruído visual para um look corporativo de alto nível.
- **Responsive Bento Grid**: Layout moderno que organiza informações em cards interativos com animações "Spring" (GSAP).
- **Internacionalização (i18n)**: Suporte total para **Português, Inglês e Espanhol**.

## 🛠️ Stack Tecnológica

- **Core**: HTML5, CSS3 (Vanilla), JavaScript (ES6+).
- **Animações**: [GSAP](https://greensock.com/gsap/) (ScrollTrigger, Spring physics).
- **3D Engine**: [Three.js](https://threejs.org/).
- **Data Viz**: [Chart.js](https://www.chartjs.org/) & GitHub API.
- **Scroll**: [Lenis](https://github.com/darkroomengineering/lenis) (Smooth scroll).
- **Ícones**: FontAwesome & DevIcons.

## 📁 Estrutura do Projeto

- `index.html`: Estrutura semântica e SEO.
- `style.css`: Design system e variáveis de tema.
- `script.js`: Lógica do Bruno-bot, i18n e manipulação do DOM.
- `three-scene.js`: Motor do cenário 3D Neural Flow.
- `certificates/`: Repositório de certificações acadêmicas.
